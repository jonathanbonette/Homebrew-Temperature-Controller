\doxysubsection{Detailed Description}
Definição da interface de callbacks para a máquina de estados Itemis. 

Esta classe implementa as operações de callback definidas no modelo Itemis. Ela serve como uma camada de abstração entre a lógica da máquina de estados e as operações de baixo nível do hardware, como controle de display, leitura de teclado, controle de PWM e comunicação com outras tarefas Free\+RTOS. \begin{DoxyAuthor}{Author}
