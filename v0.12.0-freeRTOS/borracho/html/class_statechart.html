sc_boolean Statechart::isActive ( ) const
virtual

Checks if the state machine is active (until 2.4.1 this method was used for states). A state machine is active if it has been entered. It is inactive if it has not been entered at all or if it has been exited.

Implements sc::StatemachineInterface.
